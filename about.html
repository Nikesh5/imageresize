How to work image resizer

The web page should load and display the image resizer. To use the image resizer, click on the "Choose Image" button and select an image file from your local machine. The selected image will be displayed in the "Original Image" box. Enter the desired size (in pixels) for the resized image in the "Image Size" input field, and click on the "Resize Image" button. The resized image will be displayed in the "Resized Image" box. To download the resized image, click on the "Download Resized Image" button. The resized image will be downloaded to your local machine with the filename "resized-image.png". Note that this is a simple example of an image resizer and may not work with all types of images or in all web browsers.
